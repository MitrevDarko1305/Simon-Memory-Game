# Simon Memory Game 🎮

A browser-based memory game inspired by the classic Simon Game.

The game generates a sequence of colors and sounds, and the player must repeat the sequence correctly. Each level adds a new step to the pattern and increases the difficulty.

## 🚀 Features

* Interactive color sequence
* Sound effects for each button
* Increasing speed as levels progress
* Mobile and desktop support
* Game over animation

## 🛠 Built With

* HTML
* CSS
* JavaScript
* jQuery

## 🎯 How to Play

1. Press any key or tap the title to start.
2. Watch the color sequence carefully.
3. Repeat the sequence by clicking the buttons.
4. Each level adds a new color to remember.
5. If you click the wrong button, the game ends.
